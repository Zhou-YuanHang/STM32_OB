Obsidian 引用其他文件主要分两大类：**链接（跳转）**与**嵌入（内容直接显示）**，核心是 `[[ ]]` 和 `![[ ]]`。下面按常用场景给你最全、最实用的写法。
## 一、基础链接（跳转，不显示内容）
### 1. 链接到文件（Wiki 语法，默认）
```markdown
[[目标笔记]]          # 同目录，省略 .md
[[目标笔记.md]]        # 带后缀也可以
[[文件夹/目标笔记]]    # 子目录
```

### 2. 带别名（显示文字≠文件名）
```markdown
[[目标笔记|点击查看]]
```
显示为：**点击查看**，点击跳转到目标笔记。
### 3. 链接到标题（锚点）
```markdown
[[目标笔记#章节标题]]
[[目标笔记#一级标题#二级标题]]
```

直接跳转到指定标题位置。

### 4. 链接到块（精准到段落 / 列表）

先给目标块加 ID（行尾加 `^xxx`）：
```markdown
这是一段内容 ^block-01
```
然后链接：
```markdown
[[目标笔记#^block-01]]
```
常用于**跨笔记引用单段内容**。
### 5. Markdown 原生链接（兼容外部）
```markdown
[显示文字](目标笔记.md)
[显示文字](文件夹/目标笔记.md)
```
需手动处理空格（%20），一般不用。

---
## 二、嵌入文件（内容直接显示，同步更新）
在链接前加 `!`，内容**内联显示**，源文件修改后自动更新 Obsidian。
### 1. 嵌入整篇笔记
```markdown
![[目标笔记]]
```
### 2. 嵌入指定标题下内容
```markdown
![[目标笔记#章节标题]]
```
### 3. 嵌入单个块
```markdown
![[目标笔记#^block-01]]
```
### 4. 嵌入图片 / 附件
```markdown
![[图片.png]]
![[附件/PDF文件.pdf]]
```
桌面端可**直接拖放文件**自动生成嵌入代码 Obsidian。
---
## 三、高级用法

### 1. 链接 / 嵌入到 heading + 块
```markdown
[[目标笔记#标题#^block-id]]
![[目标笔记#标题#^block-id]]
```
### 2. 别名 + 嵌入（少用）
```markdown
![[目标笔记|别名]]  # 显示别名，内容仍嵌入
```
### 3. 反向链接（自动）
只要用 `[[ ]]` 引用，目标笔记右侧**反向链接面板**会自动显示被哪些笔记引用。

## 四、速查表（复制即用）

| 功能   | 语法                   |         |
| ---- | -------------------- | ------- |
| 基础链接 | `[[笔记名]]`            |         |
| 别名链接 | `[[笔记名               | 显示文字]]` |
| 标题锚点 | `[[笔记名#标题]]`         |         |
| 块引用  | `[[笔记名#^block-id]]`  |         |
| 嵌入全文 | `![[笔记名]]`           |         |
| 嵌入标题 | `![[笔记名#标题]]`        |         |
| 嵌入块  | `![[笔记名#^block-id]]` |         |

## 五、小技巧
- 重命名文件时，Obsidian **自动更新所有链接**（设置可关）Obsidian。
- 链接失效用 **Link Doctor** 检查修复。
- 大量引用优先用**块 ID**，比标题更稳定（标题易改）。
