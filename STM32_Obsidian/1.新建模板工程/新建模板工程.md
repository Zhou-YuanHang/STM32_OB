# 1. 添加启动文件
启动文件以 `.s` 结尾
```
芯片上电 / 复位
    │
    ▼
① 硬件从 0x0000_0000 读取 4 字节 → 装入 SP（栈顶指针）
    │
    ▼
② 硬件从 0x0000_0004 读取 4 字节 → 装入 PC（程序计数器）
    │   这个地址就是 Reset_Handler 的入口
    ▼
③ 跳转到 Reset_Handler 执行
    │
    ▼
④ 调用 SystemInit()
    │   配置 HSE → PLL → 72 MHz 系统时钟
    ▼
⑤ 跳转到 C 库 __main
    │   ① 拷贝 .data 段（Flash → RAM）
    │   ② 清零 .bss 段
    │   ③ 初始化堆
    ▼
⑥ 调用你写的 main()
    │
    ▼
⑦ 进入 while(1) 无限循环
```

## 4.1 寄存器开发过程
+ 肯定是 GPIOC打开时钟，
+ 配置 PC13 这个端口的输出模式
+ 设置高低电平
```C
#include "stm32f10x.h"                  // Device header

int main(void)
{
	RCC->APB2ENR=0x00000010;
	GPIOC->CRH = 0x00300000;
	
	while(1)
	{
		
		GPIOC->ODR = 0x00002000;
		//GPIOC->ODR = 0x00000000;
		
	}
	
}

```
