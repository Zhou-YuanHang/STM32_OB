# AHB 总线上挂载着什么
![[assets/Pasted image 20260615230935.png]]
从这个图上来看的话，AHB 只挂载了 APB1和 2
但是实际上 SRAM，DMA 都挂载在 AHB 总线上，这点去查看参考手册的 AHB 外设时钟使能寄存器(RCC_AHBENR)也能验证这点。
___
所以我觉得这点也可以解释为什么不需要去手动打开 AHB 的时钟，因为 AHB 挂载着 SRAM 这种外设，没有 SRAM 系统肯定无法运行啊，所以 AHB 的时钟是自动打开的。
其实这里的表示有个误区，不是打开时钟，而是**使能挂在 APB2 上的 GPIOC 这个外设的门控时钟**，这点去看时钟树可以得到验证，所以准确的说是 APB2 的时钟 PCLK2 已经存在了，现在需要用户使能门控才能将 PCLK2 送给 APB2
[具体解释](使能GPIO时钟为什么只需配置APB2.md)

# 寄存器，存储器和寄存器映射，存储器映射
寄存器可以理解为一些列高速存储单元，CPU 要是想操作这些寄存器就必须知道他们的地址，这就需要用到寄存器映射了。
存储器和存储器映射也是类似的道理，只是存储器是一个完整的区域，里面包括了寄存器。
![[assets/Pasted image 20260623225446.png|625]]
```C
// 示例：写一个外设寄存器（伪代码）
*(volatile uint32_t*)(0x40020014) = 0x0000FF00;  // 直接往地址写值
```
